HTML

HTML adalah singkatan dari Hypertext Markup Language.

HTML tetapi bahasa markup. Markup languages adalah bahasa komputer yang digunakan untuk menyusun, memformat, atau mendefinisikan hubungan antara berbagai bagian dokumen teks dengan bantuan simbol atau tag yang disisipkan dalam dokumen.

Dalam HTML Terdapat beberapa Text Formatting

  • text tebal (bold)
  • text penting (strong)
  • text miring (italic)
  • text menekankan (emphasis)
  • text menandai (mark)
  • text garis bawah (underline)
  • text kecil (small)
  • text mencoret (delete)
  • text sisipan (insert)
  • text pangkat dibawah (subscript)
  • text pangkat di atas
